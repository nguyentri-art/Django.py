create virtual environment
    -python -m venv venv
Then activate that environment
    -venv\Scripts\activate

// nếu chưa cài django  và pip 
Then install Django in that environment

    -pip install django

Then create django project named mysite

    -django-admin startproject mysite

Khi bắt đầu tạo database
sử dụng để cập nhật

Use python manage.py makemigrations

sau đó python manage.py migrate --run-syncdb
